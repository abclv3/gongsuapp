## 📲 **Android (삼성, LG 등) 설치 방법**

### 1단계: 웹사이트 접속
1. **Chrome 브라우저** 열기
2. 주소창에 입력: `https://gongsuapp.vercel.app`
3. 웹사이트 열림

### 2단계: 홈 화면에 추가
1. 우측 상단 **⋮ (점 3개)** 버튼 클릭
2. **"홈 화면에 추가"** 선택
3. 이름 확인: **Safety-Pay**
4. **"추가"** 버튼 클릭

### 3단계: 완료! 🎉
- 홈 화면에 **Safety-Pay 아이콘** 생성됨
- 아이콘을 누르면 앱처럼 실행!
- 브라우저 없이 전체 화면으로 사용 가능!

### 3단계: 완료! 🎉

## 🎓 **처음 사용하기**

### 1️⃣ 회원가입 (최초 1회만)
1. 앱 실행
2. **"계정이 없으신가요? 회원가입"** 클릭
3. 정보 입력:
   - **아이디**: 4자 이상
   - **비밀번호**: 6자 이상 (꼭 기억하세요!)
   - **이름**: 본명
   - **휴대폰번호**: 010-1234-5678
   - **입사년월**: 입사한 년월 선택
   - **현장**: 
     * **"GPS 자동탐지"** 클릭 (위치 권한 허용)
     * 또는 직접 검색 (예: "청라" 입력)
4. **"회원가입 완료"** 클릭

### 2️⃣ 로그인
1. 아이디 입력
2. 비밀번호 입력
3. **"로그인"** 클릭

### 3️⃣ 급여 계산하기
1. **급여 귀속월** 선택
2. **내 공수** 입력 (근무한 일수)
3. **입사일자** 확인/수정
4. **공제 유형** 선택:
   - 사업소득세 3.3%
   - 4대보험 9.4%
5. 자동으로 **실수령액** 계산됨!

### 4️⃣ 월차 관리
- **만근 (26일)** 달성 시 자동으로 월차 1일 발생
- 상단 **녹색 배지** 클릭 → 월차 내역 확인
- **"+ 월차사용"** 버튼으로 월차 사용

## 🆘 **자주 묻는 질문 (FAQ)**

### Q1: 앱이 아니라 웹사이트인가요?
**A**: PWA(프로그레시브 웹 앱)으로, 웹사이트를 앱처럼 사용할 수 있습니다!  
홈 화면에 추가하면 일반 앱과 똑같이 작동합니다.

### Q2: "홈 화면에 추가" 메뉴가 안 보여요
**A**: 
- **Android**: Chrome 브라우저 사용 확인
- **iPhone**: Safari 브라우저 사용 확인 (Chrome X)

### Q3: 데이터가 안전한가요?
**A**: 모든 데이터는 **본인 휴대폰**에만 저장됩니다.  
서버에 개인정보가 전송되지 않아 안전합니다.

### Q4: GPS 권한을 꼭 허용해야 하나요?
**A**: 선택사항입니다!  
현장을 자동으로 찾으려면 필요하지만, 수동으로 검색해서 선택할 수도 있습니다.

### Q5: 로그인이 자꾸 풀려요
**A**: 브라우저를 닫으면 보안상 로그인이 해제됩니다.  
회원가입 정보는 저장되어 있으니 다시 로그인하시면 됩니다.

### Q6: 비밀번호를 잊어버렸어요
**A**: 현재는 비밀번호 찾기 기능이 없습니다.  
팀장님께 문의하여 새 계정을 만들어주세요.

### Q7: 인터넷 없이 사용할 수 있나요?
**A**: 
- **로그인/회원가입**: 인터넷 필요 ❌
- **급여 계산**: 오프라인 가능 ✅
- **데이터 저장**: 오프라인 가능 ✅

### Q8: 휴대폰을 바꾸면 데이터가 사라지나요?
**A**: 네, 데이터는 휴대폰에 저장되므로 새 휴대폰에서는 다시 가입하셔야 합니다.
