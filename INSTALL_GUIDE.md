# 📱 Safety-Pay 앱 설치 가이드

## 🎯 안전감시단 팀원 여러분께

Safety-Pay를 **앱처럼 설치**해서 사용하실 수 있습니다!  
**APK 다운로드 필요 없이**, 웹사이트에서 바로 설치하세요!

---

## 📲 Android (삼성, LG 등) 설치 방법

### Step 1: 웹사이트 접속
1. **Chrome 브라우저** 열기
2. 주소창에 입력:  
   👉 `https://safety-pay.vercel.app` (배포 후 실제 URL 제공)

### Step 2: 홈 화면에 추가
1. 우측 상단 **⋮ (점 3개)** 클릭
2. **"홈 화면에 추가"** 선택
3. **"추가"** 버튼 클릭

### Step 3: 완료! 🎉
- 홈 화면에 **Safety-Pay 아이콘** 생성됨
- 앱처럼 실행 가능!

---

## 🍎 iPhone (iOS) 설치 방법

### Step 1: 웹사이트 접속
1. **Safari 브라우저** 열기 (중요!)
2. 주소창에 입력:  
   👉 `https://safety-pay.vercel.app`

### Step 2: 홈 화면에 추가
1. 하단 **공유 버튼** (⬆️) 클릭
2. **"홈 화면에 추가"** 선택
3. **"추가"** 버튼 클릭

### Step 3: 완료! 🎉
- 홈 화면에 **Safety-Pay 아이콘** 생성됨
- 앱처럼 실행 가능!

## ✨ 설치 후 장점

### 📱 앱처럼 사용
- **전체 화면**: 브라우저 주소창 없이 전체 화면
- **빠른 실행**: 아이콘 터치로 즉시 실행
- **알림바에 고정**: 앱 전환 시 표시

### 🔒 안전하고 편리

- **자동 업데이트**: 앱 업데이트 자동 적용

### 🚀 빠른 성능
- **GPS 기능**: 현장 자동 감지 (HTTPS 보안)
- **빠른 로딩**: 캐시로 빠른 실행

## 🆘 문제 해결

### Q1: "홈 화면에 추가" 메뉴가 안 보여요
**A**: 
- Android: Chrome 브라우저 사용 확인
- iPhone: Safari 브라우저 사용 확인 (Chrome X)

### Q2: 설치 후 아이콘이 안 보여요
**A**: 
- 홈 화면을 좌우로 스와이프하여 확인
- 앱 서랍(App Drawer)에서 확인

### Q3: GPS 기능이 작동 안 해요
**A**: 
- 휴대폰 설정 → 위치 권한 허용
- Chrome/Safari에서 위치 권한 허용

### Q4: 로그인이 풀려요
**A**: 
- 정상입니다. 보안상 세션 종료 시 재로그인 필요
- 회원가입 정보는 저장되어 있습니다

## 🎓 간단 사용법

### 1️⃣ 회원가입 (최초 1회)
- 아이디, 비밀번호 설정
- 이름, 휴대폰번호 입력
- 입사년월 입력
- 현장 선택 (GPS 자동 또는 수동)

### 2️⃣ 로그인
- 아이디 + 비밀번호 입력

### 3️⃣ 급여 계산
- 귀속월 선택
- 공수 입력 (근무 일수)
- 자동 계산 결과 확인!

### 4️⃣ 월차 관리
